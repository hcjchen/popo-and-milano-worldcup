# 🐱⚽ Popo & Milano World Cup Daily Quiz Shootout

A cute, single-page football game starring **Popo** (orange-and-white cat striker) and **Milano** (tuxedo-cat goalkeeper). Answer 5 daily World Cup trivia questions — every correct answer earns Popo a 10-second penalty round to beat Milano!

## ▶️ Play now

**https://hcjchen.github.io/popo-and-milano-worldcup/**

Works on computers and phones. The same 5 questions appear for everyone each day, so you can compare scores with friends.

## How to play

1. Type your name.
2. Answer each question within **10 seconds**. Harder questions are worth more points (Easy 100 · Medium 150 · Hard 250).
3. Each correct answer unlocks a **10-second penalty round** — aim with your mouse/finger, hold to charge power, release to shoot past Milano.
4. After 5 questions, see your score, your grade, and share it with friends.

**Score** = quiz points + 50 per goal.

## Features

- 150+ verified World Cup trivia questions (FIFA history, hosts, winners, legends, team nicknames, 2026 facts, and more)
- Daily challenge: same questions for everyone each day
- Difficulty tiers (Easy / Medium / Hard) with higher rewards for harder questions
- Penalty shootout with a charging power meter and a diving keeper
- Best score saved on your device
- Mobile + desktop friendly, no installs

## Tech

A single self-contained `index.html` — vanilla HTML5 Canvas + JavaScript, no dependencies, no build step. Hosted free on GitHub Pages.
